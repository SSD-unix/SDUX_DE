# SDUX Desktop Environment

SDUX Desktop Environment is a custom Linux desktop shell written in Python using PyQt.  
It is being developed as a core component of the upcoming SDUX Operating System.

This project represents the graphical and user interaction layer of SDUX OS.

## üöÄ About This Project

The SDUX Desktop Environment is not just a GUI application.  
It is designed to:

- Manage sessions
- Launch and control applications
- Provide an integrated terminal
- Include a lightweight file manager
- Embed a web browser (Qt WebEngine)
- Explore interaction with Linux system components (D-Bus, X11)

This project is part of a long-term goal:  
to build a custom operating system environment from the ground up.
